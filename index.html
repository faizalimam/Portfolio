Manuscript dalam Aplikasi (Mandala)

as Programmer

Mandala is an ancient Sundanese manuscript image directory application, with features for visualize ancient Sundanese manuscripts, non-script image galleries, scientific papers related to Ciburuy's site research, games, transliteration, indexation, and search engines. This desktop application is a form of cultural preservation devoted to the existing script in Ciburuy site. This application was developed using visual studio 2015 with C # programming language



In addition to having a page for visitors, in this application there is a page for the admin. the purpose of the admin page is to manage the datababase that contains data about the site photo or data about the corpse. The database used is MySql.
